# Skill — FAQ Horaires (exemple)

Réponse standard quand quelqu'un demande les horaires / jours d'ouverture.

## Déclencheurs

Intentions / mots-clés : `horaires`, `ouvert`, `ouverture`, `fermé`, `demain`, `ce week-end`,
`quelle heure`, `dispo`.

## Variables (à adapter)

- HORAIRES = "du lundi au vendredi, 9h–18h"
- FERMETURE = "le week-end et les jours fériés"

## Réponse type

> Bonjour {prénom} 👋 On est ouverts {HORAIRES}. C'est fermé {FERMETURE}.
> Je peux faire autre chose pour vous ?

## Ton

Chaleureux, tutoiement OU vouvoiement selon le message reçu, une emoji max.

## Ne pas déclencher si

- La personne demande un rendez-vous précis / une dispo à une date donnée → c'est un autre cas,
  ne pas répondre automatiquement (laisser l'humain).
